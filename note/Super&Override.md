# Super

- 基本介绍：

    - `super`表示父类的引用，用于访问父类的**属性**，**方法**，和**构造器**

- 基本语法

    1. 访问父类的属性，但不能访问父类的private属性

     ```java
        super.属性名；
     ```

    2. 访问父类的方法，不能访问父类的private方法
    ```java
    super.方法名(形参列表)
    ```

    3. 访问父类的构造器
	```java
	super.(参数列表)；//这里只能放在构造器的第一句，只能出现一句
